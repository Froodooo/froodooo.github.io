--- title: From literature notes to permanent notes layout: page comments: false ---

Process of taking (literature) notes and transfering them to permanent notes (based on Zettelkasten).

  1. Read an article, watch a video, read a book. Anything you can write notes for.
  2. While consuming, write notes in your own words (no copy-pasting) into a new file tagged with type "note" and the type of what you consumed (e.g. video, book, article, etc.)
  3. From the notes file, take one or more atomic ideas and write a file on it, using your own thoughts and words. Tag the file as "permanent".
  4. (Optional) Link the permanent file from a MOC.
