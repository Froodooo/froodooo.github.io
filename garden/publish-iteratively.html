--- title: Publish Iteratively layout: page comments: false ---

Programming is an iterative process. You build software, release it, receive feedback and then restart the cycle. The same idea can be applied to publishing writings. You could call it Test-Driven Writing (based on the development term Test-Driven Development). You deploy writings instead of publishing them.
